Задание основано на ДЗ к 5 уроку.

Задание 1
Убрать кнопки «Свернуть», «Развернуть», «Свернуть всё», «Развернуть всё». Убрать поле добавления комментариев. Убрать все обработчики, связанные с этими элементами.

Задание 2
Изменить список постов, используя виджет accordion из jQuery UI.

Задание 3
Добавить поле ввода с автокомплитом и кнопку «Поиск». В качестве элементов автокомплита взять заголовки всех постов. При клике на кнопку «Поиск» в списке остаётся только искомый пост. Для автокомплита использовать виджет autocomplete из jQuery UI.
